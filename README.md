# IoT Smart Air Quality Monitoring System

This project monitors air quality using an MQ135 gas sensor and uploads real-time data to a cloud platform via ESP8266 WiFi. It allows users to track pollution levels remotely using web or mobile dashboards.

## Features
- Real-time air quality monitoring  
- WiFi-based cloud data upload  
- Remote access via IoT platform  
- Useful for pollution detection and safety  

## Hardware Required
- ESP8266 (NodeMCU)  
- MQ135 Gas Sensor  
- WiFi connection  
- Power supply  

## Applications
- Smart cities  
- Indoor air monitoring  
- Industrial safety
